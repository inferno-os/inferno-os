lucishell - Read-only access to the Lucishell terminal

Usage:
  read [body]     Read the shell transcript
  read input      Read the current input line
  tail [n]        Read the last n lines of transcript (default 30)
  status          Check if shell is running

NOTE: This tool is READ-ONLY for safety.
You cannot send commands to the shell.
Instead, propose commands in Chat for the user to run.

The shell must be running. Use 'launch lucishell' to start it.

Examples:
  lucishell read              Read the full transcript
  lucishell tail              Last 30 lines
  lucishell tail 50           Last 50 lines
  lucishell read input        Current input line
  lucishell status            Check shell status
